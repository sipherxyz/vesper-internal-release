# Vesper Releases

Pre-built binaries for Vesper. See the [Releases](https://github.com/sipherxyz/vesper-internal-release/releases) page for downloads.

## Platforms

- **macOS** (Apple Silicon): `Vesper-arm64.dmg`
- **macOS** (Intel): `Vesper-x64.dmg`
- **Windows**: `Vesper-x64.exe`
- **Linux**: `Vesper-x86_64.AppImage`
